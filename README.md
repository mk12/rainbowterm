# Rainbowterm

Rainbowterm is a tool for managing color schemes in iTerm2.

## Install

Install with [Homebrew][hb]:

    brew tap mk12/rainbowterm https://github.com/mk12/rainbowterm
    brew install mk12/rainbowterm/rainbowterm
